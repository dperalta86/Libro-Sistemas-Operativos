# Recursos Adicionales

## Bibliografía Recomendada

Para profundizar en teoría:  
- Silberschatz, Galvin, Gagne. *Operating System Concepts* (10ª ed.). El clásico académico, excelente para fundamentos teóricos.  
- Tanenbaum, Bos. *Modern Operating Systems* (4ª ed.). Más orientado a implementación, con buenos ejemplos.  

Para programación de sistemas:  
- Stevens, Rago. *Advanced Programming in the UNIX Environment* (3ª ed.). La biblia para syscalls y programación en Unix/Linux.  
- Kerrisk. *The Linux Programming Interface*. Exhaustivo y actualizado específicamente para Linux.  

Para internals del kernel:  
- Love. *Linux Kernel Development* (3ª ed.). Accesible introducción al desarrollo del kernel.  
- Bovet, Cesati. *Understanding the Linux Kernel* (3ª ed.). Más detallado sobre implementación interna.
